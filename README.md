# Learning Management System

## Requirements

- Node.js (recommended: latest LTS)
- npm (comes with Node.js)

## Setup

```bash
npm install
```

## Run (development)

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build & run (production)

```bash
npm run build
npm run start
```

## Lint

```bash
npm run lint
```
